# Lineflow

![Screen Recording 2023-12-18 at 2 44 30 PM](https://github.com/leomcelroy/lineflow/assets/27078897/0333b4ca-52cd-4bca-9141-44d05d2775ce)

Lineflow is a tool for making animated single line drawings.

It was built for an MIT museum exhibit.

It's intended to be used with a pen plotter like the Hack Club Blot

# Usage

First draw a single line drawing in the app.

You must start in the left red dot and end in the right red dot.

<img width="1364" alt="Screenshot 2023-12-14 at 3 01 43 PM" src="https://github.com/leomcelroy/lineflow/assets/27078897/349afdd0-7447-4d55-8dbc-1e777b8da730">

Hit Animate

<img width="114" alt="Screenshot 2023-12-14 at 2 56 37 PM" src="https://github.com/leomcelroy/lineflow/assets/27078897/e3c089e8-357b-4285-ad06-1e3a18c9889f">

Copy the path by clicking the path button to open the text window.

<img width="1233" alt="Screenshot 2023-12-14 at 2 56 46 PM" src="https://github.com/leomcelroy/lineflow/assets/27078897/6f525700-33ec-45ab-b844-aa97690eb8e2">

Paste the path into the blot script given in `examples/blot.js`.

<img width="1426" alt="Screenshot 2023-12-14 at 3 02 02 PM" src="https://github.com/leomcelroy/lineflow/assets/27078897/50e219ef-3cf6-44d2-9ab2-91661ab2d666">
